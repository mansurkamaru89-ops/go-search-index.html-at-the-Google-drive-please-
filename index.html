# 🔐 Login Hub A simple web-based login system built using HTML, CSS, and JavaScript. ## 🚀 Features - ✅ Email whitelist system - 📊 Login logs (email + time + status) - 🚨 Suspicious activity detection - 🔒 System lock after multiple failed attempts - 🧠 Basic input protection (anti-swear & anti-injection) - 📋 Dashboard to view logs ## 🛠️ How it works - Only allowed emails can access the dashboard - All login attempts are logged - Multiple failed attempts trigger alerts and lock the system ## ⚠️ Note This is a **frontend demo project**: - No real database - Uses browser localStorage - Logs are stored per device only ## 🌐 Live Website (Your GitHub Pages link will go here) ## 👨‍💻 Author Mansur Kamaru
